# 金種計算機＋ サポートページ

## アプリについて

金種計算機＋は、レジ締めや現金管理を簡単にする金種計算アプリです。
数式入力、画像共有、自由なカスタマイズ機能を備えています。

## 主な機能

### 数式入力
- 電卓のように数式を入力できます(例: 100+50+20)
- 複雑な枚数計算も瞬時に処理

### 画像共有
- 集計結果を画像として保存・共有できます
- LINE、メール、Slackなどで簡単に報告可能

### カスタマイズ
- 10色以上のカラーテーマから選択可能
- お札・硬貨の表示モード(画像/テキスト/併用)を選択可能

## プライバシーポリシー

### 収集する情報
本アプリは、以下の情報のみを扱います:

- **アプリ設定情報**: カラーテーマや表示モードなどの設定(端末内のみに保存)
- **広告ID**: Google AdMobによる広告配信のため

### 収集しない情報
本アプリは以下の情報を**収集しません**:

- 個人を特定できる情報
- 入力された金額や枚数データ
- 位置情報
- 連絡先情報
- カメラやマイクへのアクセス

### データの保存
- すべての入力データは端末内のみに保存されます
- サーバーへのデータ送信は行いません
- アプリを削除すると、すべてのデータが削除されます

### 広告について
本アプリは Google AdMob を使用して広告を表示しています。
AdMobは広告配信のために以下の情報を使用する場合があります:

- 広告ID
- IPアドレス
- デバイス情報

詳細は [Google のプライバシーポリシー](https://policies.google.com/privacy) をご確認ください。

## よくある質問

### Q. データはどこに保存されますか?
A. すべてのデータは端末内のみに保存されます。クラウドやサーバーへの送信は行いません。

### Q. 入力した金額データは共有されますか?
A. いいえ、入力データは一切外部に送信されません。

### Q. 画像共有機能は何を共有しますか?
A. 集計結果の画像のみを共有します。画像は端末内で生成され、共有先はユーザーが選択します。

### Q. アプリを削除するとデータはどうなりますか?
A. アプリを削除すると、すべての設定とデータが完全に削除されます。
